truST Web IDE Preview

This preview exposes a secure API baseline for remote Structured Text editing. Use /api/ide/capabilities and /api/ide/session to bootstrap a client.

Authoring writes are intentionally gated behind runtime debug mode and optimistic version checks to avoid accidental production edits.

UI editing widgets are not shipped in this phase. MP-043 currently provides API contracts, auth/session hardening, and conflict-safe document operations.
